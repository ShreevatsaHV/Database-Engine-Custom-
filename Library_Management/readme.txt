The operating system used to develop the system is Windows 10.

The application requirements are

MAMP : To run the PHP files and connect to the SQL database. 
MAMP version 3.2.2 is to be installed(follow the instructions on the official website). 
Once installed, all the files that has to be executed is to be present in the htdocs folder within the MAMP folder(including html, js files along with PHP files). 
If any images are required in the interface, those have to be present in htdocs folder as well.

For development of the above mentioned files, Eclipse IDE is used.
